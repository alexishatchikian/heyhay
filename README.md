## Getting Started

This repository hosts the HeyHay project - a modern Armenian learning application built with React, TypeScript, Tailwind CSS, and Framer Motion.

### Development

```bash
npm start
```

Runs the app in development mode at [http://localhost:3000](http://localhost:3000).

### Build

```bash
npm run build
```

Builds the app for production to the `build` folder.
